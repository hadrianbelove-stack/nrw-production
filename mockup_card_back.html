Design Notes:

  • CURRENT: Shows STREAM/RENT/BUY - generic actions, doesn't show WHERE to watch
  • PROPOSED: Shows platform names (AMAZON, APPLE TV, NETFLIX) - clearer for users
  • Button Logic: Collects all platforms from streaming/rent/buy, deduplicates (Amazon appears in both rent+buy = one button)
  • Priority: Netflix > Amazon > Apple > Disney+ > Hulu > HBO Max > Others
  • Max Buttons: Show top 3 platforms (configurable)
  • Colors:
    • Amazon = Orange (#ff9900) - matches Amazon branding
    • Apple = Gray (#555) - matches Apple TV branding
    • Netflix = Red (#e50914) - matches Netflix branding
    • Disney+ = Blue (#113ccf) - matches Disney+ branding
    • Others = Generic blue (#007aff)
  • Affiliate Links: All Amazon links include ?tag=yoursite-20, Apple links include ?at=yourtoken
  • Click Behavior: Clicking any button opens that platform's page where user can rent/buy/stream

Alternative Option - Hybrid Approach:

  • Keep buttons labeled as actions: RENT, BUY, STREAM
  • Add small platform label/icon above or inside button: Amazon
  • Example: Button says "RENT" with small "via Amazon" text
  • Pros: Clearer action (rent vs buy), still shows platform
  • Cons: More complex UI, less space on small cards
